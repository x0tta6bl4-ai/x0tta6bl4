## Когда можно использовать архивные документы?

- **Исторический контекст:** Понимание эволюции планов

**НЕ использовать для:**
- ❌ Планирования текущей работы
- ❌ Оценки статуса компонентов
- ❌ Определения приоритетов
- ❌ Коммуникации со стейкхолдерами
