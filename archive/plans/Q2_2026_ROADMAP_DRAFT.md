# Roadmap x0tta6bl4 (Q2 2026) - Draft

**Статус:** На обсуждении (Проект достиг TRL 8, все планы Q1 выполнены).

## Стратегические направления

### 1. Web3 Экономика (Mainnet Integration)
Перевод финансового контура из тестового режима в боевой.
- [ ] Развертывание смарт-контрактов X0T в L2 сети (Base / Arbitrum).
- [ ] Интеграция `TokenBridge` с реальными RPC-провайдерами (с использованием Circuit Breakers).
- [ ] Автоматизация конвертации Fiat-to-Crypto (Stripe -> USDC -> X0T) для скрытия блокчейн-сложности от конечных B2B пользователей.

### 2. Доступность для пользователей (Mobile & Edge)
Расширение клиентской базы за счет удобных интерфейсов.
- [ ] Разработка Mobile SDK (Android/iOS) с поддержкой PQC-туннелей (Kyber).
- [ ] Создание легковесного клиента для OpenWRT (роутеры).
- [ ] Релиз десктопного PWA приложения на базе Tauri/Electron.

### 3. Глубокая безопасность (Hardware Root of Trust)
Усиление Zero-Trust архитектуры.
- [ ] Интеграция аттестации Intel SGX / AMD SEV для Exit-нод.
- [ ] Аппаратное хранение PQC-ключей (TPM/HSM) на критических узлах.
- [ ] Внедрение обфускации трафика (TrafficObfuscator) в eBPF-контур по умолчанию.

### 4. Масштабирование интеллекта (Swarm Evolution)
Развитие интеграции Kimi K2.5.
- [ ] Обучение GraphSAGE на реальных данных телеметрии (за Q1).
- [ ] Внедрение динамического ценообразования в Marketplace на основе предсказаний спроса (RL).
- [ ] Интеграция мультимодальных агентов для автоматического парсинга скриншотов от пользователей (Support Agent).
