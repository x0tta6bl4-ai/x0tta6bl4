# NL Templates

This directory is for redacted examples only.

Never store production values here:

```text
UUIDs
private/public keys
short IDs
passwords
bot tokens
subscription URLs
x-ui web base path
```

Candidate templates:

```text
nl-beta-2443.example.json
x-ui-config-shape.example.json
```

Created templates:

```text
nl-beta-2443.example.json
```

`nl-beta-2443.example.json` is a shape template only. It preserves the current
NL beta inbound structure: VLESS Reality on port 2443, one example client, three
example server names, one example short ID, and direct outbound. All production
UUID/key/shortId/serverName values are replaced with examples or placeholders.
