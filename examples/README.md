# Examples

This directory contains a collection of examples demonstrating how to use the x0tta6bl4 framework. Each example is designed to showcase specific features and use cases.

---

## Examples

### 1. Example Complete Integration

This example demonstrates complete integration of x0tta6bl4.

File: `example_complete_integration.py`

Run it:
```bash
python examples/example_complete_integration.py
```

### 2. EBFP Performance Monitoring

This example demonstrates how to use the eBPF performance monitoring features of x0tta6bl4.

File: `examples/ebpf_performance_monitoring.py`

Run it:
```bash
python examples/ebpf_performance_monitoring.py
```

### 3. AI Gateway Demo

This example shows how to use x0tta6bl4 as a self-healing AI gateway with post-quantum cryptography and zero-trust security.

Directory: `examples/ai-gateway`

Run it:
```bash
cd examples/ai-gateway
docker-compose up -d
```

### 4. Integration with Existing Python Project

This example demonstrates how to integrate x0tta6bl4 into an existing Python project. It shows how to use the self-healing mesh architecture with post-quantum cryptography in an existing Flask application.

File: `examples/example_integration_with_existing_project.py`

Run it:
```bash
python examples/example_integration_with_existing_project.py
```

## Running Examples

### Prerequisites

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   pip install -r requirements-dev.txt
   ```

2. Configure your environment:
   - Follow the instructions in the main README.md file
   - Set up any required credentials or configuration files

### Running an Example

1. Navigate to the project root:
   ```bash
   cd /path/to/your/project
   ```

2. Run the example:
   ```bash
   python examples/example_name.py
   ```
