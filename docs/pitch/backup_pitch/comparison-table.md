# Сравнительная таблица оборудования

## 1. Orange Pi 5 Plus (4 шт.)

| Характеристика | Orange Pi 5 Plus | Raspberry Pi 5 (8GB) | Orange Pi 5 (8GB) |
|:---------------|:----------------:|:--------------------:|:-----------------:|
| **RAM** | 16 GB LPDDR5 | 8 GB LPDDR4X | 8 GB LPDDR5 |
| **Процессор** | RK3588 (8 ядер, 4xA76+4xA55) | BCM2712 (4 ядра A76) | RK3588S (8 ядер) |
| **Сеть** | 2.5GbE | Gigabit Ethernet | Gigabit Ethernet |
| **Шина расширения** | M.2 M-Key (PCIe 3.0 x4) | PCIe 2.0 x1 | M.2 M-Key |
| **Цена** | **12 990 ₽** (onpad.ru) | ~12 000 ₽ | ~8 000 ₽ |
| **Почему выбрана** | 16GB RAM для mesh-узлов, 2.5GbE для реальной нагрузки на сеть | Не хватает RAM для криптографии | Меньше RAM, нет 2.5GbE |

## 2. MikroTik CSS326-24G-2S+RM

| Характеристика | MikroTik CSS326 | MikroTik CRS305 | D-Link DGS-1100 |
|:---------------|:---------------:|:---------------:|:----------------:|
| **Порты** | 24×1GbE + 2×10GbE SFP+ | 4×10GbE SFP+ | 8×1GbE |
| **Управление** | Smart Switch (L2+) | L3 | L2 |
| **QoS/эмуляция сбоев** | Port storm control, ACL | Полный L3 | Базовый |
| **Цена** | **17 000 ₽** (DNS) | ~18 000 ₽ | ~6 000 ₽ |
| **Почему выбран** | Достаточно портов для стенда + SFP+ для внешней сети | Не хватает медных портов | Слишком базовый, нет SFP+ |

| **Каналы** | 16 | 8 | 2 |
| **Частота** | 200 MS/s | 500 MS/s | 48 MS/s |
| **Протоколы** | SPI, I2C, UART, CAN | SPI, I2C, UART, CAN | SPI, I2C, UART |
| **Цена** | **~37 000 ₽** (chipdip) | ~35 000 ₽ | ~5 000 ₽ |
| **Почему выбран** | 16 каналов, доступен в РФ, достаточная частота | 8 каналов, дороже | Слишком медленный |
