## Реализованные проекты

### Платформа защищённых распределённых сетей (x0tta6bl4)
*2024-2026*

Production-ready платформа с постквантовой криптографией и обработчики сетевых пакетов dataplane. Полный стек: от обработки пакетов на уровне ядра Linux до интерфейса управления.

**Статус:** v0.2.0-rc1, 925K+ строк кода, 1312 тестов, CI/CD.

**Ключевые компоненты:**
- PQC стек: современное шифрование-768 + современные подписи-65 (чистый Python, без зависимостей)
- обработчики сетевых пакетов dataplane: 142k PPS TX, 49k PPS RX
- Self-healing: MAPE-K цикл, автоматическое восстановление
- Identity: SPIFFE/SPIRE, JWT-SVID prod mode
- CI/CD: GitHub Actions, readiness gate (25 проверок)

### Стандарт защищённой связи для беспилотных аппаратов (x0tMQ)
*2025-2026*

Разработка открытого протокола для постквантового шифрования каналов управления БПЛА. Спецификация опубликована в формате технический стандарт Internet-Draft.

**Результат:** Спецификация 540 строк, код 3300+ строк, бенчмарки на ARM64.

### Криптографическое ядро (современное шифрование-1024 + современные подписи-87)
*2025-2026*

Реализация постквантовых алгоритмов шифрования — общепризнанных мировых эталонов стойкости (международные эталоны стойкости 203/204/205). Устранены уязвимости, связанные с утечкой данных по времени выполнения.

**Результат:** 3300+ строк кода, все тесты проходят.
