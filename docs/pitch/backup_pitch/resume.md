## Реализованные проекты

### Платформа защищённых распределённых сетей (x0tta6bl4)
*2024-2026*

Production-ready платформа с постквантовой криптографией и высокопроизводительный eBPF dataplane. Полный стек: от обработки пакетов на уровне ядра Linux до интерфейса управления.

**Статус:** v0.2.0-rc1, 925K+ строк кода, 1312 тестов, CI/CD.

**Ключевые компоненты:**
- PQC стек: шифрование ML-KEM-768 и подписи ML-DSA-65 (чистый Python, без зависимостей)
- высокопроизводительный eBPF dataplane: 142k PPS TX, 49k PPS RX
- Self-healing: MAPE-K цикл, автоматическое восстановление
- Identity: SPIFFE/SPIRE, JWT-SVID prod mode
- CI/CD: GitHub Actions, readiness gate (25 проверок)

### Стандарт защищённой связи для беспилотных аппаратов (x0tMQ)
*2025-2026*

Разработка открытого протокола для постквантового шифрования каналов управления БПЛА. Спецификация опубликована в формате технический стандарт Internet-Draft.

**Результат:** Спецификация 540 строк, код 3300+ строк, бенчмарки на ARM64.

### Криптографическое ядро (шифрование ML-KEM-768 и подписи ML-DSA-65)
*2025-2026*

Реализация постквантовых алгоритмов шифрования — общепризнанных мировых эталонов стойкости (стандарты FIPS 203/204/205). Устранены уязвимости, связанные с утечкой данных по времени выполнения.

**Результат:** 3300+ строк кода, все тесты проходят.
