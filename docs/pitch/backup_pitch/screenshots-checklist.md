# Скриншоты цен — чек-лист для заявителя

Нужно сделать **1 скриншот на каждую позицию** (с любой площадки: DNS, Ozon, Яндекс Маркет).
Желательно 3 площадки на каждую — если есть время.
Сохранить в папку `docs/pitch/screenshots/`.

## Где брать скриншоты

| Площадка | Поиск | Как сохранить |
|:---------|:------|:-------------|
| **dns-shop.ru** | Ввести название → отсортировать по цене → скриншот результата | `Print Screen` → вставить в Paint → сохранить |
| **ozon.ru** | Поиск → скриншот карточки товара с ценой | Скриншот страницы товара |
| **market.yandex.ru** | Поиск → скриншот выдачи с ценами | Скриншот результата поиска |

## Что должно быть на скриншоте (минимально)

1. **Название товара** (чтобы было видно что это то самое)
2. **Цена** (крупно)
3. **Дата** (видна в строке браузера или в углу скрина)
