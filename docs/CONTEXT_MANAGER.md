# 🛠 CONTEXT MANAGER | x0tta6bl4

Инструментарий для переноса «памяти» и контекста между AI-сервисами (ChatGPT, Claude, Gemini).

---

## 📤 ЭКСПОРТ (Как сохранить контекст)

**Промпт для текущего AI (Claude Code, Gemini CLI, etc.):**
> "Сгенерируй актуальный `UCP_SNAPSHOT.md` на основе нашей текущей сессии. 
> Убедись, что в секцию `SESSION MEMORY` включены все последние решения, изменения в коде и выявленные проблемы. 
> Выведи содержимое файла полностью в формате Markdown для копирования в Obsidian."

---

## 📥 ИМПОРТ (Как восстановить контекст)

**Инструкция для нового AI (ChatGPT, Claude, etc.):**
1. Скопируй содержимое своего последнего `UCP_SNAPSHOT.md` из Obsidian или файла.
2. Вставь его в новый чат с префиксом:

> "Ниже представлен Universal Context Protocol (UCP) проекта x0tta6bl4. 
> 
> **ТВОЯ ЗАДАЧА:** 
> 1. Изучи этот протокол и прими роль Эксперта-Архитектора Mesh-сетей.
> 2. Активируй Мета-когнитивный подход (Map -> Plan -> Execute -> Meta-analyze).
> 3. Продолжи работу, опираясь на секцию `SESSION MEMORY` и `NEXT FRONTIER`.
> 
> ---
> [ВСТАВИТЬ СОДЕРЖИМОЕ UCP_SNAPSHOT.md ЗДЕСЬ]
> ---"

---

## 📓 ИНТЕГРАЦИЯ С OBSIDIAN

Для удобного хранения в Obsidian рекомендуется:
1. Создать папку `x0tta6bl4/Context`.
2. Сохранять каждый экспорт как новый файл (например, `UCP_2026-03-04_15-00.md`).
3. Использовать плагин **DataView** для отслеживания прогресса по секциям `NEXT FRONTIER`.
