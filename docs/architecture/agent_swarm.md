# Agent Swarm Architecture (Kimi K2.5)

## Архитектура
x0tta6bl4 использует концепцию роя агентов для обеспечения автономности и самоисцеления. 

Ядром системы является **PARL (Parallel Agent Reinforcement Learning) Engine**, который позволяет выполнять до 1500 параллельных шагов рассуждения агентов.

### Ключевые компоненты
1. **SwarmOrchestrator**: Управляет жизненным циклом агентов.
2. **PARLController**: Диспетчеризует задачи на воркеров (AgentWorker) с использованием алгоритмов PPO (Proximal Policy Optimization).
3. **Agent Capabilities**: Каждый агент имеет строго ограниченный набор разрешений (Zero-Trust), проверяемый через `AntiMeaveOracle`.
4. **Vision Module**: Позволяет агентам "видеть" состояние системы через анализ скриншотов топологии и UI-дашбордов (Coding with Vision).

### Интеграция с MAPE-K
Рой агентов интегрирован в фазы Analyze и Plan цикла MAPE-K. Когда детектируется аномалия, `PARLMAPEKExecutor` распараллеливает анализ возможных причин (например, Causal Analysis) среди свободных агентов, сокращая MTTR.
