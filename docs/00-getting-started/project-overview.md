# x0tta6bl4 v1.5.0-alpha

**Самовосстанавливающаяся децентрализованная mesh-платформа** с автоматическим консенсусом, распределённым хранилищем и Zero Trust безопасностью.

> **Текущая версия:** v1.5.0-alpha  
> **Статус:** 🟢 Production Ready (95%+)  
> **Релиз:** 7 ноября 2025 г.

---

## 🎯 Что это?

x0tta6bl4 — это **единственная в России** open-source платформа для создания самовосстанавливающихся mesh-сетей с:
- ✅ **Автоматическим консенсусом** (Raft) — узлы сами выбирают лидера
- ✅ **Распределённым хранилищем** (CRDT + KVStore) — данные синхронизируются автоматически
- ✅ **Zero Trust безопасностью** (SPIFFE/SPIRE) — каждый процесс имеет цифровой паспорт
- ✅ **Самолечением** (MAPE-K) — система сама чинит себя при сбоях
- ✅ **Высокой производительностью** (eBPF + Batman-adv) — пакеты летают на уровне ядра

**Для кого:** государственные структуры, телеком-операторы, крупные корпорации, DevOps-инженеры.
