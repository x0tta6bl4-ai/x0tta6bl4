# ISO/IEC 27001 Statement of Applicability (SoA)

Version: 0.1  
Last updated: 2026-03-04  
Status: Working draft for readiness program

## Applicability Rules

- Applicability is evaluated for x0tta6bl4 SaaS + staging operations.
- Status values:
  - `Implemented`: control evidence exists and is operating.
  - `Partial`: evidence exists but not yet complete/approved/periodic.
  - `Planned`: control is required but not yet implemented as an auditable process.
