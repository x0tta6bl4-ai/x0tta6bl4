# VPN System x0ttta6bl4

## Обзор

Система VPN x0ttta6bl4 предоставляет высокий уровень анонимности и защиты от утечек трафика. Она включает в себя множество функций для обхода блокировок, защиты от слежки и безопасного интернет-сurfing.

## Ключевые функции

### Защита от утечек
- **DNS Leak Protection**: Предотвращает утечки DNS-запросов через зашифрованные DNS-over-HTTPS соединения
- **IP Leak Detection**: Проверяет соответствие публичного IP-адреса ожидаемому VPN-адресу
- **WebRTC Leak Protection**: Предотвращает утечки через WebRTC протокол
- **Kill Switch**: Автоматически блокирует трафик при потере VPN-соединения
- **Firewall Configuration**: Настраивает firewall для блокировки всех не-VPN трафика

### Обфускация трафика
5 методов обфускации для обхода блокировок:
- **FakeTLS**: Имитирует TLS 1.3 трафик
- **ShadowSocks**: Шифрует с использованием ChaCha20-Poly1305
- **Domain Fronting**: Скрывает истинный адрес за доменами CDN
- **StegoMesh**: Кодирует трафик в HTTP/ICMP/DNS пакетах
- **Hybrid**: Комбинирует несколько методов для максимальной защиты

### Формирование трафика
Имитация трафика популярных приложений:
- YouTube/Netflix (видеотрансляция)
- WhatsApp/Signal (голосовые вызовы)
- Веб-браузинг (HTTPS)
- Загрузка файлов
- Игры (низкая задержка)

### Ротация параметров
- **SNI**: 18 популярных CDN и доверенных доменов
- **TLS Fingerprints**: Chrome, Firefox, Safari, Edge, iOS, Android
- **SpiderX Paths**: 15 реалистичных HTTP-путей
- **Стратегии ротации**: Временная, случайная, круговая

## Мониторинг и управление

### Скрипты управления
1. **vpn_system_status.py** - Полный обзор статуса системы
2. **start_vpn_protection.py** - Включение всех функций безопасности
3. **stop_vpn_protection.py** - Выключение защиты и восстановление конфигурации
4. **vpn_connection_check.py** - Быстрая проверка соединения и соответствия IP
5. **test_vpn_anonymity.py** - Комплексный тест всех компонентов

### Пример использования
```bash
# Проверить статус системы
python3 vpn_system_status.py

# Включить защиту
python3 start_vpn_protection.py

# Проверить соединение
python3 vpn_connection_check.py

# Выключить защиту
python3 stop_vpn_protection.py
```

## Конфигурация

### Переменные окружения
- `X0TTA6BL4_SHADOWSOCKS_PASSWORD`: Пароль для шифрования ShadowSocks
- `DNS_RESOLVERS`: Настраиваемый список DNS-серверов
- `OBFUSCATION_METHOD`: Метод обфускации (faketls, shadowsocks, domain_fronting, stegomesh, hybrid)

### Порт
- **Стандартный**: 39829 (общий доступ)
- **Экспериментальный**: 39830 (оптимизирован для обхода блокировок)

## Производительность

- **DNS-запросы**: ~150-300 мс
- **Обфускация**: 10-19x увеличение размера
- **Формирование трафика**: <50 мс дополнительной задержки

## Безопасность

1. **Закрытый трафик**: Все соединения через VPN-интерфейс
2. **Зашифрованные DNS**: DNS-over-HTTPS с конфиденциальными резольверами
3. **Брандмауэр**: Блокировка не-VPN трафика
4. **Kill Switch**: Защита при потере соединения
5. **Обфускация**: Скрытие типа трафика
