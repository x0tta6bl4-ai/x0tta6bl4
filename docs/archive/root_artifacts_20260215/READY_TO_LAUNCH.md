### Шаг 1: Настроить

```bash
# Быстрый setup
./QUICK_SETUP.sh

# Или вручную
pip install -r requirements_sales.txt
cp .env.example .env
nano .env  # Заполнить токен и кошельки
```

### Шаг 2: Заполнить .env

```bash
TELEGRAM_BOT_TOKEN=your_token_from_botfather
USDT_TRC20_WALLET=TYourWalletAddress
TON_WALLET=UQYourTonWalletAddress
```

### Шаг 3: Запустить

```bash
python3 src/sales/telegram_bot.py
```

### Шаг 4: Протестировать

1. Открой Telegram
2. Найди бота

4. "Смотреть тарифы" → Проверь цены
5. Выбери тариф → Проверь адреса
