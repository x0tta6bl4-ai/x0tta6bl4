# x0tta6bl4 Public Node Program (PNP)

Become a part of the world's first self-healing, PQC-secured mesh network. 
Contribute your infrastructure and earn rewards in the MaaS ecosystem.

## Goals
- **Global Coverage**: Reach 100+ countries with low-latency anchor nodes.
- **Resilience**: Decentralize the control plane across community-owned hardware.
- **Privacy**: Ensure zero-knowledge traffic routing for all users.

## How It Works
1. **Provision Hardware**: Any Linux server (VPS, Raspberry Pi, Industrial PC) with 1GB+ RAM.
2. **Install Agent**: Run the one-line installer with the `--public` flag.
3. **Earn Rewards**: Receive **Governance Power (GP)** and **Service Credits** for every hour of stable uptime.
