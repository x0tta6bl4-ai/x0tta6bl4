# Welcome to x0tta6bl4 Documentation

This documentation provides a comprehensive overview of the x0tta6bl4 project, from high-level architecture to detailed API references.

## What is x0tta6bl4?

x0tta6bl4 is the world's first mesh VPN with **real post-quantum cryptography**. It provides a secure, resilient, and decentralized communication layer for the post-quantum era.
