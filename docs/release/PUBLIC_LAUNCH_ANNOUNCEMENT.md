# 🚀 x0tta6bl4 v1.1.0 "Empirical Integrity" is LIVE!

We just released x0tta6bl4 v1.1.0 — the first Zero-Trust Mesh Network that values **Engineering Honesty** over hype.

### What's inside:
- 🛡️ **NIST-Compliant PQC:** ML-KEM & ML-DSA protection for all transit.
- ⚡ **Verified eBPF Datapath:** 142k TX PPS baseline on physical hardware.
- 📶 **5G Core Interop:** Verified SCTP signaling & User Plane against Open5GS.
- 🆔 **Supply Chain Security:** Keyless release signing via Sigstore/Rekor.

### Why "Empirical Integrity"?
During our final RC1 audit, we discovered a simulation error that claimed 8.8M PPS. Instead of "optimizing" it, we **purged** it. We chose to start our GA with a honest, verified baseline of 142k PPS. 

Because in infrastructure, **Integrity is the only protocol that matters.**
