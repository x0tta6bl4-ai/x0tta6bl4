# Ответ на вопрос: "Почему проект retired?"

## Вопрос работодателя

> *«Почему вы остановились? Что помешало довести до stage-окружения? Это не выглядит как незавершёнка или потеря интереса?»*

## Ответ (для собеседования)

**Кратко:**  
Проект не заброшен. Он перешёл из стадии «single-node VPS» в «local Docker cluster» из-за потери production-сервера. Разработка продолжается.

**Детально:**

В июне 2026 закончилась аренда VPS (89.125.1.107) — единственного сервера, который был выходом в мир. Бюджет на продление отсутствовал, поэтому всё production-окружение (SPIRE, mesh-node, Ghost Access, x-ui) было свёрнуто и пересобрано в локальный Docker-стенд на домашней машине.

Что произошло с проектом:

| Было (VPS) | Стало (локально) |
|---|---|
| Один сервер NL (89.125.1.107) | 2 mesh-node + SPIRE cluster в Docker |
| SPIRE server/agent на хосте | SPIRE server + agent в контейнерах |
| Ghost Access production | Ghost VPN server + Redis (Docker, healthy) |
| Mesh-node production | mesh-node-a + mesh-node-b (Docker, connected) |
| Self-healing daemon | Self-healing daemon (systemd, работает) |

Это **не потеря интереса**, а адаптация к реальности: zero-budget, санкции, соло-разработка. Кодовая база жива — последний коммит сегодня (1033 коммита, активная работа). Docker-стенд полностью функционален: SPIRE, mesh-ноды, Ghost VPN — всё работает и тестируется.

**Что я сделал правильно:**  
С самого начала проектировал систему так, чтобы она не зависела от конкретного сервера. SPIRE + Docker позволили переехать на локальную машину за один день. Если появится новый сервер — деплой занимает час.

**Что я извлёк:**  
- Единственная точка отказа (single VPS) — риск для проекта 
- Контейнеризация с самого начала — правильное решение
- Следующий этап: деплой на bare-metal или mesh из нескольких локальных машин

**Про долгосрочную мотивацию:**  
Проект идёт 1.5 года, 1033 коммита — это не вспышка интереса, а устойчивый ритм. Я не строю стартап. Я строю инфраструктуру, которая работает независимо от внешних обстоятельств. Потеря VPS — не конец, а смена формы существования.
