🔍 DDoS Анализ

Определим источник атаки, тип вектора и рекомендации по защите. Не "DDoS обнаружен", а "атака идёт с X.X.X.X через UDP flood на порт 53, блокировать Y.Y.Y.Y/24".

📊 Network Forensics

Реконструируем что произошло в сети. Кто с кем общался, какие данные передавались, были ли утечки. Уровень: kernel-space.

⚡ Anomaly Detection

Найдём аномалии, которые не видит обычный мониторинг. eBPF видит каждый пакет — пропустить невозможно.
