## Подготовка к демо

### Перед демо
1. Проверить, что система работает
2. Запустить port-forward (если нужен)
3. Подготовить credentials
4. Открыть документацию

### Что показать
1. Health check
2. Mesh network status
3. Peers list
4. Routes
5. Metrics (если доступны)

---

## Сценарий 1: Basic Health Check

**Цель:** Показать, что система работает

**Шаги:**
1. Открыть терминал
2. Выполнить:
   ```bash
   curl http://localhost:8080/health
   ```
3. Показать ответ:
   ```json
   {
     "status": "ok",
     "version": "3.4.0-fixed2",
     "timestamp": "2026-01-08T05:45:00Z"
   }
   ```

**Объяснение:**
- Система работает
- Версия видна
- Timestamp показывает актуальность

**Время:** 1-2 минуты

---

## Сценарий 2: Mesh Network Status

**Цель:** Показать состояние mesh сети

**Шаги:**
1. Выполнить:
   ```bash
   curl http://localhost:8080/mesh/status
   ```
2. Показать ответ:
   ```json
   {
     "status": "healthy",
     "nodes": 5,
     "links": 10,
     "uptime": "24h 30m"
   }
   ```

**Объяснение:**
- Mesh сеть работает
- Количество узлов и связей
- Время работы

**Время:** 2-3 минуты

## Сценарий 3: Peers List

**Цель:** Показать список пиров

**Шаги:**
1. Выполнить:
   ```bash
   curl http://localhost:8080/mesh/peers
   ```
2. Показать ответ:
   ```json
   {
     "peers": [
       {
         "id": "node-1",
         "status": "connected",
         "latency": "10ms"
       },
       {
         "id": "node-2",
         "status": "connected",
         "latency": "15ms"
       }
     ]
   }
   ```

**Объяснение:**
- Список подключенных узлов
- Статус каждого узла
- Задержка связи

**Время:** 2-3 минуты

## Сценарий 4: Routes

1. Выполнить:

- Маршруты между узлами
- Путь пакетов

**Время:** 2-3 минуты

## Сценарий 6: API Token Usage

**Цель:** Показать, как использовать API token

**Шаги:**
1. Показать token из `api_token.txt`
2. Выполнить:
   ```bash
   curl -H "Authorization: Bearer YOUR_TOKEN" \
        http://localhost:8080/api/v1/status
   ```
3. Показать ответ

**Объяснение:**
- Как использовать token
- Безопасность
- API endpoints

**Время:** 2-3 минуты
