Deploy in One Command

Kubernetes 1.28+ required. Helm 3.14+.

# Add Helm OCI registry
helm registry login registry.gitlab.com \
  -u <user> -p <token>

# Deploy enterprise release
helm upgrade --install x0tta \
  oci://registry.gitlab.com/x0tta/charts/x0tta6bl4-commercial \
  --version 1.0.0 \
  --namespace x0tta-production \
  --create-namespace \
  --values values-enterprise.yaml \
  --wait --timeout=15m

Or kubectl apply -f production/all-in-one.yaml  |  Full install guide (AWS/GCP/Azure/bare metal)
