# Compliance Documentation

This directory contains the current compliance-readiness package for x0tta6bl4.

## ISO/IEC 27001 Readiness Pack

- [ISO/IEC 27001:2025 readiness report](ISO_IEC_27001_2025_READINESS.md)
- [Statement of Applicability (SoA)](ISO_27001_2025_SOA.md)
- [Evidence index](ISO_27001_2025_EVIDENCE_INDEX.md)
- [Risk treatment plan](ISO_27001_2025_RISK_TREATMENT_PLAN.md)

## Usage

1. Use the readiness report as the baseline status.
2. Use the SoA as the control mapping source of truth.
3. Use the evidence index during audit preparation and evidence collection.
4. Use the risk treatment plan to track remaining gaps to audit readiness.
