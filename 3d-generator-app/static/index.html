Как пользоваться

  1. Выбери шаблон или впиши код вручную
  2. Опиши что нужно в любом AI (ChatGPT, Claude, Gemini) — попроси сгенерировать build123d код
  3. Вставь код в редактор
  4. Нажми "Сгенерировать" — получи STL/STEP

Промпт для AI

Сгенерируй build123d Python код для: [опиши модель].
Используй from build123d import *.
Результат положи в переменную shape.
Используй параметрические размеры.
