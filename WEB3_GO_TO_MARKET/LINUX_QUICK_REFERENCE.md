## **БЫСТРЫЕ ОТВЕТЫ НА ВОПРОСЫ:**

### "Как это работает?"
→ eBPF программы на XDP обрабатывают пакеты до kernel network stack. Self-healing через GraphSAGE ML.

### "Сколько времени на setup?"
→ 1-2 дня для 100 узлов (vs 2-4 недели с Ansible).

### "Нужен ли Docker?"
→ Нет, есть нативные пакеты (.deb, .rpm, .apk).

### "Работает на старом Linux?"
→ Да, если kernel >= 4.9 (Ubuntu 18.04+, Debian 9+).

### "Какое влияние на производительность?"
→ Ускоряет: 0.1-1ms (vs 10-50ms без eBPF). CPU overhead: ~2-5%.

### "Нужны root права?"
→ Для eBPF — да, но можно через capabilities (без полного root).
